Smarter Way to Learn Chapters 41-50

Chapter 41: While loops

The while loop loops through a block of code as long as a specified condition is true.

  • var counter is initialized with a value before the loop.
  • The counter is then updated within the code.

Chapter 42: Doing...While loops

The do...while statement creates a loop that executes a block of code, before checking if the condition is true or false, then it will repeat the loop as long as the condition is true.

Chapter 43: Placing scripts appropriatly

JavaScript code is inserted between <script> and </script> tags.

You can place multiple scripts in an HTML document. Scripts can be placed only in the <body>, or in the <head> section of an HTML page, or even in both.

Chapter 44: Commenting

Comments can be used to explain JavaScript code to make it easier to read and understand.

Single line comments start with //

Multi-line comments start with /* and end with */.

Chapter 45: Events with a Link

HTML events are changes that happen to the HTML elements. When JavaScript is used in HTML pages, JavaScript can alter or move on these events. HTML allows event handler attributes to be added to HTML elements.

onClick: is the keyword that executes the browser to watch for a click on a link.

Chapter 46: Events with a button

The code creates a button that on click, executes a function.

<input type="mybutton" value="Click me" onClick="Something();">

Chapter 47: Events with a mouse

onMouseOver: The user moves the mouse over an HTML element.

onMouseOut: The user moves the mouse away from an HTML element.

Chapter 48: Events with a field

onFocus: is the keyword for detecting that the user has clicked in a field

onBlur: is the keyword for detecting that a field no longer has the focus

Chapter 49: Reading fields values
