Smarter Way to Learn JS Chapters 1-10

Chapter 1: Alerts

An alert box is often used if you want to make sure information comes through to the user. When an alert box pops up, the user will have to click "OK" or "Cancel" to proceed.

Chapter 2: Variables for Strings

Variables are used to store data values like Strings. Heres my example "The variable name is "PC" and the value assigned is " PC Master Race"."

Chapter 3: Variables for Numbers

Variables are used to store data values like Numbers. Numbers do not need quotation marks like strings need to display.

Chapter 4: Variable Names that are Legal and Illegal

Variables cannot be named with the reserved JavaScript keywords.

Legal variable name: dollar sign ($), underscore (_), letters, and numbers.

Numbers cannot be at the beginning of a variable name.

Chapter 5: Math Expressions with familiar operators

Javascript perform basic math operations like addition (+), subtraction (-), multiplication(*), division(/) and modulus (%).

var number = 100;

var division = 4;

var result = number / division;

Chapter 6: Math Expressions with unfamiliar operators

JavaScript has the options to Increase or Decrease only one number from the variable.

Chapter 7: Math Expressions and eliminating ambiguity

JavaScript has rules that it will, first solve multiplication and division, then addition and subtraction. Use parenthesis to specifiy which operation you want to do first.

Try using the following equation: 10-5*5.

Chapter 8: Concatenating text strings

Concatenate text strings by adding separated strings together by using + between them.

Chapter 9: Prompts

Prompt is used to get user input. The structure is typically: prompt(question, [default answer]);

Chapter 10: if statements

If statements are very useful when checking for a given condition.

Change the "Yes!" to anything else to get another answer.
