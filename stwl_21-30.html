Use indexOf method is used to find specific segments in a string.

Chapter 24: Strings and finding a character at a location

Use charAt for finding a character at a specific location in a string.

Chapter 25: Strings and replacing characters

Use replace for removing one set of characters and inserting another set in its place.

Chapter 26: Rounding numbers up and down

This chapter shows different ways to round numbers.

Use Math.round to round to the nearest integer.

Use Math.ceil to round up.

Use Math.floor to round down.

Chapter 27: Generating random numbers

Use Math.random() to generate random numbers.

Chapter 28: Converting strings to integers and decimals

Use parseInt to convert a numerical string to a number data type.

Use parseFloat to conver a decimal string to a decimal number type to reserve the decimal places.

Chapter 29: Converting strings to numbers, numbers to strings

Use Number() to conver strings to numbers.

Use toString() to convert numbers to strings.

Use toFixed() to control the length of decimals.
