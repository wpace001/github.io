Smarter Way to Learn JS Chapters 11-20

Chapter 11: Comparison operators

These are the comparison operators: ===, !==, >, and <. You can compare al the different data types using the comparison operators such as these.

Chapter 12: if...else and else if statements

Else will also test for a condition when all the previous tests have failed.

Chapter 13: Testing sets of conditions

To test more than one condition using the if statement you must add (&&) two conditions; or use the || to test if either condition is true.

Chapter 14: if statements that are nested

An alternative to using && to test multiple conditions in a single if statement is to use something like a nested if statement.

Chapter 15: Arrays

An array holds any number of values like a database. In an array, the numbering starts with zero. An example would be something like an Excel spreadsheet.

Chapter 16: Arrays and adding and removing elements

POP removes the last element from an array.

PUSH adds one or more elements to the end of an array.

Chapter 17: Arrays and removing, inserting, and extracting elements

SHIFT removes first element of an array.

UNSHIFT adds one or more elements to the beginning of an array

SPLICE inserts new elements anywhere in an array and/or removes elements from an array

SLICE copies elements from an array to another array.

Chapter 18: for regular loops

Loops are very useful when you want to run the same code again and again, with a different value.

Chapter 19: for loops with Flags, Booleans, Array Length, and lastly Loopus Interruptus

BREAKs stops a loop from continuing to loop.
