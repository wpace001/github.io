Smarter Way to Learn Chapters 31-40

Chapter 31: Current date and time

Date() Creates a new date object to get the current date and time

Chapter 32: Extracting date and time

The Date() object can be modified to get different time format.

getDay(): The day from the Date object.

getMonth(): The month from the Date object.

getDate(): The date from the Date object.

getFullYear(): The year from the Date object.

getHours() : The hour from the Date object.

getMinutes(): The minute from the Date object.

getSeconds(): The second from the Date object.

getMillisecond(): The millisecond from the Date object.

getTime() : The milliseconds since: December 7, 1941.

Chapter 33: Specify the date and time

var todayDay = new Date("MonthDate,Year");

Chapter 34: Altering the elements of date and time

Using the below methods to change parts of a Date object.

setFullYear(): The year to the Date object.

setMonth(): The month to the Date object.

setDate(): The date to the Date object.

setHours(): The hour to the Date object.

setMinutes() : The minute to the Date object.

setSeconds(): The second to the Date object.

setMilliseconds(): The milliseconds past the second of the Date object.

getMillisecond(): The millisecond from the Date object.

getTime() : The milliseconds since: December 7, 1941.

Chapter 35: Functions

A function is a block of code that has a specific task and can be reused when the time is needed.

Chapter 36: Functions: Passing data

A function can be set up to allow users to give data when it is asked for.

Chapter 37: Functions: Passing data back to them

A function can return a value to the code that asks for it.

Chapter 38: Functions: Local vs. Global variables

A variable declared inside the main code is a global variable and known globally by all functions in that code.

A variable declared inside a function is a local variable and is known only inside that function.
